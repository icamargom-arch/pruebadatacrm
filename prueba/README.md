
# Prueba Técnica – Node.js MVC (Sin Framework)

Este proyecto implementa:

✔ MVC   
✔ Node.js
✔ jQuery para consumo AJAX  
✔ Tabla con diseño profesional  
✔ Logs y validaciones  
✔ Estructura senior  

## Instalación
node version 22.21.1
npm install descargar module



```bash
node server.js
npm start correr proyecto
```

Abrir:  
http://localhost:3000
